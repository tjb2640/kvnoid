# 20/26/02/16

Initial format version

## Header
- 4B Magic `KVNF`
- 4B Version code
- 16B UUID (major/minor bytes)
- 8B Epoch MS date created
- 8B Epoch MS date modified
- 24B Reserved
- 4B Length of category == C
- 4B Length of nametag == N
- 4B Length of serialized key bytes == K (in body)
- 4B Length of encrypted value == V (in body)
- 52B Reserved
- (C)B Category
- (N)B Nametag
- 8B JLong CRC32 up to this point (excl. paddings)
- 4B padding

## Body
- (K) Serialized `AESGCMKey` bytes
- (V) Encrypted data
- 4B padding
- Variable length-mod-4 null block padding to end
