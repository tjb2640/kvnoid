# 20/26/02/16

Initial format version

## Header
- 4B Magic `KVNF`
- 4B Version code
- 16B UUID (major/minor bytes)
- 8B Epoch MS date created
- 8B Epoch MS date modified
- 24B Reserved
- 4B Length of category key Kc
- 4B Length of category C
- 4B Length of nametag key Kn
- 4B Length of nametag N
- 4B Length of value key Kv
- 4B Length of encrypted value V
- 52B Reserved
- 8B JLong CRC32 up to this point (excl. paddings)
- 4B padding

## Body
- (Kc)
- (C)
- (Kn)
- (N)
- (Kv)
- (V)
- 4B padding
- Variable length-mod-4 null block padding to end

## Considerations
While category and nametag are encrypted at rest,
it's not so important to me that they remain strictly
obfuscated while in memory, even if they are stored
as ObfuscatedString.
